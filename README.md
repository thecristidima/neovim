External tools that need to be installed to make everything work:
- `git`
- `pwsh` (Windows-only)
- `rg`
- `fd`
- `fzf`
- `lazygit`
- `win32yank`
- `node` / `npm`
- `.NET SDK` / `dotnet`
- `roslyn-language-server` - via Mason or as a .NET global tool:
  `dotnet tool install -g roslyn-language-server --prerelease`
